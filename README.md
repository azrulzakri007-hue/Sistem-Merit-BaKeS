# Sistem Merit Pengawas

Sistem ini dibangunkan bagi tujuan mengurus dan merekodkan markah merit pengawas sekolah. 
Setiap pengawas akan bermula dengan jumlah merit asas sebanyak **100 markah**. 
Sekiranya berlaku kesalahan disiplin, markah merit akan ditolak berdasarkan tahap kesalahan yang telah ditetapkan.

Sistem ini menggunakan pelayar web (browser) dan menyimpan data secara automatik melalui `localStorage`. 
Ini bermaksud data akan kekal selama pelayar tidak dipadamkan.

---

## 1. Objektif Sistem

1. Menyediakan platform digital yang mudah diakses bagi pengurusan merit pengawas.
2. Menggantikan rekod manual kepada sistem yang lebih sistematik dan teratur.
3. Memudahkan pemantauan markah merit secara masa nyata.
4. Mengurangkan risiko kehilangan rekod merit.

---

## 2. Fungsi Utama Sistem

- Menambah nama pengawas baharu.
- Memberikan markah permulaan sebanyak 100 merit kepada setiap pengawas.
- Menolak markah merit pengawas berdasarkan kesalahan tertentu.
- Menyimpan rekod merit secara automatik.
- Paparan senarai terkini markah merit pengawas.

---

## 3. Cara Menggunakan Sistem

1. Buka laman web Sistem Merit Pengawas.
2. Masukkan nama pengawas pada ruang yang disediakan.
3. Tekan butang **Tambah** untuk mendaftarkan pengawas.
4. Untuk menolak merit, tekan nilai tolak merit yang disediakan:
   - Tolak 5 markah
   - Tolak 10 markah
   - Tolak 20 markah
5. Sistem akan mengemas kini markah secara automatik.
